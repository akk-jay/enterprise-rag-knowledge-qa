- 应用类型：聊天助手
- 创建日期：2026-06-11
- 关联知识库：[填写知识库名称]

## 工作流节点配置

### 节点 1：开始 (Start)

| 参数 | 值 |
|------|-----|
| 输入变量 | sys.query（用户问题） |

### 节点 2：知识检索 (Knowledge Retrieval)

| 参数 | 值 |
|------|-----|
| 知识库 | [填写知识库名称] |
| 检索模式 | 混合检索（向量+关键词） |
| Top-K | 3 |
| 相似度阈值 | 0.7 |
| 权重设置 | 向量 0.7 / 关键词 0.3 |

### 节点 3：条件判断 (IF/ELSE)

| 参数 | 值 |
|------|-----|
| 条件 | 检索结果数量 > 0 且 最高相似度 ≥ 0.7 |
| IF 分支 | → 节点 4（LLM 生成） |
| ELSE 分支 | → 节点 5（兜底回复） |

### 节点 4：LLM 生成

| 参数 | 值 |
|------|-----|
| 模型 | [填写模型名称，如 deepseek-chat] |
| 系统提示词 | 见 prompts/system-prompt.md |
| 用户提示词 | 见 prompts/user-prompt.md |
| 温度 (Temperature) | 0.1 |
| 最大 Token | 1024 |
| 上下文 | {{#context#}} |

### 节点 5：兜底回复 (Answer)

| 参数 | 值 |
|------|-----|
| 回复内容 | 抱歉，根据现有文档未找到相关信息。建议您：1. 尝试换个关键词提问；2. 确认该内容是否在已上传的文档范围内。 |

### 节点 6：结束 (End)

| 参数 | 值 |
|------|-----|
| 输出变量 | {{#LLM.text#}} 或 兜底回复内容 |
